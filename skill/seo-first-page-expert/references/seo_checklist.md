# SEO #1 Ranking Checklist

Use this checklist to ensure every page and the site as a whole meet the highest SEO standards.

## 1. Technical SEO
- [ ] **Canonical Tags**: Every page must have a `<link rel="canonical" href="...">`.
- [ ] **Robots.txt**: Ensure `robots.txt` is present and not blocking important pages.
- [ ] **Sitemap.xml**: Ensure all pages are listed and the file is valid.
- [ ] **Mobile Usability**: Test layout on mobile viewports. No horizontal scrolling.
- [ ] **Schema Markup**: 
  - [ ] `WebSite` schema on homepage.
  - [ ] `Organization` or `ProfessionalService` schema.
  - [ ] `BreadcrumbList` for subpages.
  - [ ] `Article` or `BlogPosting` for blog posts.
  - [ ] `FAQPage` for pages with questions.

## 2. On-Page SEO
- [ ] **Title Tag**: Primary keyword at the beginning, < 60 characters.
- [ ] **Meta Description**: Compelling, includes keyword, < 155 characters.
- [ ] **H1 Heading**: Only one per page, contains the primary keyword.
- [ ] **Subheadings (H2, H3)**: Logical hierarchy, include secondary keywords.
- [ ] **Image Alt Text**: Every image must have an `alt` attribute that describes the image and contains relevant keywords where appropriate.
- [ ] **URL Slug**: Short, descriptive, uses hyphens (e.g., `isvecte-is-arama`).
- [ ] **Keyword Density**: Natural usage of keywords and LSI (Latent Semantic Indexing) terms.

## 3. Content Strategy
- [ ] **Topical Authority**: Create "clusters" of content around a main topic.
- [ ] **Internal Linking**: Link to at least 2 other relevant internal pages per post.
- [ ] **External Linking**: Link to high-authority external sources (e.g., `migrationsverket.se`).
- [ ] **Readability**: Short paragraphs, bullet points, clear font sizes.

## 4. Google "Featured Snippet" Optimization
- [ ] **Question Headers**: Use H2/H3 for common questions (e.g., "İsveç vizesi kaç günde çıkar?").
- [ ] **Direct Answers**: Provide a clear, 40-60 word answer immediately following the question header.
- [ ] **Lists/Tables**: Use HTML tables or lists for data-heavy sections.
